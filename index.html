Экономьте время благодаря фильтрам по совпадениям свойств с отклонениями

Например, Вы хотите в похожих товарах выводить товары той же ценовой категории. Вам достаточно создать одно правило, прописать там совпадение по цене с отклонением и модуль сам подберет товары с ценой в диапазоне. При настройке по точным значениям, Вы просто не сможете этого сделать, т.к. если цена будет отличаться хотя бы на копейку, значения не подтянутся.

*Фильтр работает только по каталогам товаров, без учета SKU
